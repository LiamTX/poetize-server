# poetize-api
Api do web app "Poetize", desenvolvido com express, knex e MySQL. (V1)

##First of all configure your .env with the variable
```
API_DB=...
API_USER=...
API_PASSWORD=...
API_HOST=...
```

##Project setup
```
npm install
```

##Compiles and hot-reloads for development
```
npm run dev
```
